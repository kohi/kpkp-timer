# KPKP-Timer

ワークショップ・プレゼンテーション用の多機能タイマーWebアプリです。

> (c) RE:SML — 無断での改変・再配布を禁じます。

---

## 特徴

- タイマーを最大30個登録して順番に自動実行
- カウントダウン / カウントアップの切り替え
- ストップウォッチモード搭載
- 発表者番号を管理できる人数カウンター
- BGM・アラーム・ビープ音のカスタム音声設定
- 残り時間に応じた視覚警告（背景色変化）
- プリセット保存・CSVインポート/エクスポート
- PWA対応 — ブラウザからそのまま使えて、端末へのインストールも可能
- オフライン動作対応（Service Worker）

---

## 使い方

### 基本操作

1. **設定画面**（⚙️ ボタン）でタイマーを登録
2. メイン画面に戻り「スタート」でタイマーを開始
3. タイマーが順番に自動進行、終了時にアラームが鳴る

### キーボードショートカット

| キー | 動作 |
|------|------|
| `Space` | スタート / 一時停止 |
| `R` | リセット |
| `S` / `N` | 次のタイマーへスキップ |
| `T` | メイン画面 ↔ 設定画面 |
| `M` | ミュート切り替え |
| `D` | タイマー ↔ ストップウォッチ切り替え |
| `Ctrl + C` | 人数カウンターを手動で +1 |

---

## 設定項目

| カテゴリ | 主な設定 |
|----------|----------|
| タイマー | 名前・時間・インターバル、カウントダウン/アップ切り替え |
| 人数カウンター | 表示ON/OFF、自動カウントアップ、ラベル文字 |
| 音声 | BGM・アラーム・ビープ音のファイル指定、各ON/OFF |
| 視覚警告 | 残り時間に応じた背景色変化（秒数指定） |
| 表示 | メインタイマー・残り時間・ワーク名の表示ON/OFF |
| 壁紙・カラー | 背景画像、フォント色（メイン・サブ・カウンター） |
| レイアウト | コンパクト（PC向け）/ モバイル最適化 |
| 通知 | タイマー完了時のブラウザ通知 |

---

## プリセット・CSV

- タイマー設定はプリセットとして名前をつけてブラウザに保存可能
- CSV形式でのインポート / エクスポートに対応
- バックアップにはCSVエクスポートを推奨（ブラウザのキャッシュクリアでプリセットは消える）

---

## PWAとしてインストール

対応ブラウザ（Chrome等）では「アプリとしてインストール」ボタンからホーム画面に追加できます。
設定画面 → 詳細設定から操作できます。

---

## 外部制御（PostMessage API）

iframeで埋め込んだ際、親ページからJavaScriptで操作可能です。

```javascript
iframe.contentWindow.postMessage({ action: 'start' }, '*');
// action: 'start' | 'pause' | 'reset' | 'skip' | 'settings' | 'toggleMode'
```
